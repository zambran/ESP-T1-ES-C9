# Comunicação aos Stakeholders

Prezados stakeholders,

O desenvolvimento do aplicativo para agendamento de consultas médicas continua avançando conforme o planejamento. Entretanto, foram identificados alguns riscos que exigem acompanhamento.

O principal desafio está relacionado à integração com o sistema externo de prontuário eletrônico. Alterações recentes na API e limitações na documentação aumentaram a complexidade dessa atividade, considerada essencial para a conclusão do projeto.

Também ocorreram solicitações de mudanças no fluxo de agendamento, incluindo novas regras de negócio. Essas alterações aumentaram o esforço necessário para desenvolvimento e testes.

Para reduzir os impactos, a equipe está priorizando as funcionalidades críticas, revisando o cronograma, realizando testes contínuos da integração e mantendo comunicação frequente com os responsáveis pelo sistema externo.

Nas próximas semanas continuaremos monitorando esses riscos e informaremos qualquer alteração relevante no planejamento. Nosso objetivo permanece a entrega de uma solução estável, segura e alinhada às necessidades do projeto.
