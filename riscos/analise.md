# Análise Qualitativa dos Riscos

| Risco | Probabilidade | Impacto | Fatores Condicionantes |
|--------|---------------|---------|------------------------|
| Integração com API | Alta | Alto | Alterações externas e documentação insuficiente |
| Mudanças nos requisitos | Alta | Alto | Solicitações frequentes dos stakeholders |
| Sobrecarga da equipe | Média | Alto | Equipe reduzida |
| Atraso no cronograma | Alta | Alto | Dependência da integração |
| Defeitos no software | Média | Alto | Menor tempo disponível para testes |

## Impactos esperados

- atraso na entrega;
- aumento do retrabalho;
- maior custo do projeto;
- redução da qualidade do produto;
- insatisfação dos stakeholders.
