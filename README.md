# Projeto Agendamento de Consultas

## Descrição do projeto

Este repositório reúne os artefatos produzidos durante a atividade prática da disciplina de Inteligência Artificial Generativa aplicada à Gestão de Projetos de Software.

O cenário utilizado foi o desenvolvimento de um aplicativo móvel para agendamento de consultas médicas, contemplando cadastro de usuários, gerenciamento da agenda médica, agendamento de consultas, notificações e integração com um sistema externo de prontuário eletrônico.

## Objetivo do exercício

Aplicar técnicas de gestão de riscos utilizando uma ferramenta de Inteligência Artificial Generativa para:

- identificar riscos do projeto;
- analisar seus impactos;
- definir estratégias de resposta;
- elaborar uma comunicação para os stakeholders.
